dtree 数据树文档 - layui.dtree
下拉树的使用，并且支持dtree的全部属性！
模块加载名称：dtree
  • 

    • 【新增】标签属性新增：ul标签上新增data-value属性，用于指定下拉树的初始展示值
    • 【新增】基础属性新增：select，设置树模式为下拉树
    • 【新增】基础属性新增：selectTips，设置下拉树时的输入框提示文字
    • 【新增】基础方法新增：renderSelect，初始化下拉树
    • 【新增】基础方法新增：setSelectValue，设置下拉树的初始展示值
    • 【新增】内置方法新增：setSelectValue，设置下拉树的初始展示值
    • 【新增】内置方法新增：selectSetting，设置下拉树的初始元素
    • 【新增】内置方法新增：renderSelectDom，渲染下拉树的Dom结构
常规用法

如何使用下拉树，你可以如此这般的使用它。

    									dtree.renderSelect({
									  elem: "#selTree1",
									  width: "100%", // 可以在这里指定树的宽度来填满div
									  url: "../json/case/asyncTree1.json"
									});
    指定显示框默认文字

    你可以改变下拉树默认的提示文字。

      									dtree.renderSelect({
									  elem: "#selTree2",
									  url: "../json/case/asyncTree1.json",
									  selectTips: "我想做一个成熟的提示语"
									});
      指定下拉树的默认显示值

      你可以改变下拉树的初始值。

        									// 你可以在ul标签上指定data-value
									<ul id="selTree3" class="dtree" data-id="0" data-value="湘潭市"></ul></div>
									dtree.renderSelect({
									  elem: "#selTree3",
									  url: "../json/case/asyncTree1.json",
									  selectValue: "湘潭市", // 你可以在这里指定默认值
									  done: function(){
									    dtree.dataInit("selTree3", "001003");
									    dtree.setSelectValue("selTree3"); // 也可以在这里指定，第二个参数如果不填，则会自动读取
									  }
									});
